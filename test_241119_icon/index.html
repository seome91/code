구글아이콘, 폰트어썸(font awesome)등에서 제공하는 아이콘을 사용하는방식

폰트어썸은 굉장히 많이 사용되었음 (유료화가 되어서 지금 안씀)
SI나 어플개발에서 많이 활용됨(홈페이지는 현재 인기가 없음) 아이콘 폰트를 만들어서 코드를 입력하면, 아이콘이 나타나는 방식
가벼워서 많이 쓰임,

  • 1.구글아이콘 홈페이지에서 아이콘 셋팅하고 (두께나 스타일)랜덤아이콘 클릭(오른쪽에 코드가나옴)
  • 2.오른쪽 패널에서Static icon font(가벼움) css를 복사해서 넣어준다(한번만 넣어주면됨)
  • 3.실제 홈페이지에서 사용할 아이콘을 찾고
  • 4.해당 아이콘의 실제코드(Inserting the icon) 붙여넣음
  • 5.그걸 아이콘이 들어갈 위치에 붙여넣는다
  • 6.css에서 크기와 색상 변경 가능
